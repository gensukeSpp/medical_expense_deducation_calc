# Issue #4: AI モデルを使った解析による必要データの抜き出し（概要）

目的:
- OCR の出力 JSON を入力に、LLM を用いて氏名、クリニック名、支払金額、受診日を抜き出し、構造化 JSON として出力する。

出力例:
output_json/receipt-20260101-001-structured_data.json
{
  "name": "山田 太郎",
  "clinic": "あおばクリニック",
  "amount": 3800,
  "date": "2026-01-01"
}

注意:
- LLM の解析精度が悪い場合に備え、ログとユーザー修正インターフェース（将来的に実装）を用意する。
- 解析は停止せず継続する。例外は個別ファイル単位で捕捉して記録する。
